2020-12-01

之前提到因为喜欢 Metion 而尝试恢复更新，然而除了这一点，还有别的理由。没错，阁楼由 Wordpress 换成 Hexo 的站点了。

早在 2017 年暑假，那时候我刚开始想要搭建一个个人博客。受限于财力，了解到了 Hexo + GitHub 这个免费的方法，但最终由于没弄清楚 Hexo 的使用方法放弃了。虽然现在也还是一知半解，但至少是弄明白怎么使用 Hexo 这个东西。哦对了，现在这个阁楼是部署在阿里云 OSS 上面的。本来想用 GitHub，但是每一次 hexo d后，我的域名 DNS 解析就会出问题，导致 GitHub Pages 服务无法正常使用，只能通过重新设置 GitHub Pages 的自定义域名才能使用。谷歌一番后也找不到原因，便放弃了。记得之前在哪里看到过 OSS 上可以部署 Hexo，就稍微折腾了一下。

现在阁楼缺少的应该就是评论功能，可是我又懒得去配置第三方评论服务。嘛，反正评论也不是刚需，不过如此就会少了和友人的交流。使用 Hexo 还有一点不方便。虽然非常的便捷，所有的流程都可以在本地完成，可是我写日记往往是在熄灯后躺在床铺上完成，如是，在没有电脑的情况下更新就成了无稽之谈。待到翌日早上更新，有要把 iPad 上写好的东西导出发送到电脑上 ……总之就是非常麻烦！

转眼就到了 2020 年的最后一月，更多感慨我还是留给杂话铺子水文章吧。现在稍微构思一下内容，却发现貌似并无太多可写，这么一看来我的 2020 快要化作虚无了，干面包片都比我的 2020 更有味道。

啊，突然提到了面包片，有点饿了。

晚安。
